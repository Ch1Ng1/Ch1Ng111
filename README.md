# 🥗 Калории Тракер

Прост и удобен уеб инструмент за проследяване на дневния прием на калории. Създаден за хора, които искат да се хранят по-здравословно и да следят какво консумират.

## 🚀 Функции

- ✅ Регистрация и вход за потребители
- ✅ Добавяне на хранения с дата, име и калории
- ✅ Автоматично изчисляване на общите калории за деня
- ✅ Преглед на последните 15 хранения
- ✅ Хармоника със съвети за здравословен живот
- ✅ Препоръчани закуски, обяди и вечери със средни калории
- ✅ Тестимони от потребители
- ✅ Тъмен футър с авторски права

## 🛠️ Технологии

- PHP (сесии, MySQL)
- HTML5 & CSS3
- JavaScript (хармоника)
- MySQL база данни

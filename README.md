# Визуализация на данни с графики

Този проект предоставя набор от интерактивни графики и диаграми, изградени с HTML, CSS и JavaScript (Chart.js).

## Функционалности

- **Линейна графика**: Показва тенденции в данните с течение на времето
- **Стълбова диаграма**: Визуализира сравнение между различни категории
- **Кръгова диаграма**: Показва пропорциите на части от цялото
- **CSS графики**: Алтернативни реализации само с HTML и CSS
- **Адаптивен дизайн**: Работи добре на всички размери на екрана

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, анимации)
- JavaScript (Chart.js библиотека)
- Responsive Design

## Инсталация

1. Клонирайте или изтеглете файловете на проекта
2. Отворете `index.html` във вашия браузър
3. Няма нужда от допълнителни инсталации, тъй като Chart.js се зарежда от CDN

## Как да използвате

Просто отворете HTML файла във вашия браузър и вече можете да:
- Преглеждате различните типове графики
- Интерактивно взаимодействате с графиките (за Chart.js графиките)
- Наблюдавате анимациите при преминаване с мишката

## Персонализация

За да промените данните в графиките, редактирайте JavaScript кода в секцията `<script>`:
- Променете стойностите в масивите `data`
- Редактирайте етикетите в масивите `labels`
- Променете цветовата схема чрез редактиране на `backgroundColor` и `borderColor`
